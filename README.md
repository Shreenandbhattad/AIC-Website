# Bodhi Capital Website

## Overview

Bodhi Capital is India's first undergraduate student-managed investment fund at Ashoka University. This website serves as the official platform showcasing the team, resources, and investment activities.

**Live Website**: https://www.bodhicapital.in/

## Features

- Dynamic member directory with detailed profiles
- Google Sheets integration for content management
- Real-time data updates via Apps Script
- Responsive design for all devices
- Performance-optimized with client-side caching

## Technical Stack

- HTML5, CSS3, JavaScript
- Google Apps Script
- Google Sheets (Content Management)

## Structure

- `index.html` - Landing page
- `members.html` - Team directory and profiles
- `resources.html` - Educational resources
- `newsletter.html` - Newsletter archive
- `alumni.html` - Alumni network
